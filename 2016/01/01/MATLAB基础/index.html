MATLAB基础——个人总结

  • 清空
    close all;clc;clear;

  • grid on
    ones(size(t))

    t = linspace(0, 2*pi, 100);
    x = cos(t);
    y = sin(t);
    z = ones(size(t)); %表示生成一个和t矩阵一样的所以元素都为1的矩阵
    plot3(x, y, z)
    grid on %表示在坐标轴上画满网格

  • 坐标轴范围，标刻度

    axis([0 2pi -5 5]);
    set(gca,’xTick’,[0 0.5    pi pi 1.5pi 2pi 2.5pi 3pi 3.5pi]); % 将x轴刻度分为8个点，间隔pi/2
    set(gca,’xTickLabel’,{‘0’ ‘0.5    pi’ ‘pi’ ‘1.5pi’ ‘2pi’ ‘2.5*pi’ ‘g’ ‘3.5pi’}); % 给刚才的间隔标个数值

  • 画一部分曲线

    plot(x(1:n0.5pi/0.01),z(1:n0.5pi/0.01)); %表示第1个数值画到第50pi个点，因为50pi以后表示的就是虚数了

  • 画直线

    plot([0 2*pi],[0,0],’r’)

  • 坐标轴标签

    title(‘’);
    xlabel(‘’);
    ylabel(‘’);

  • 图例

    legend(‘cp’,’cg’)

  • 矩阵变换
    ···
    B=A(end:-1:1,:)
    %表示将A的行的顺序从尾到头排列构成B，也就是B的第一行对应A的最后一行，第二行对应A的倒数第二行，以此类推。
    ···

函数：

  • find(abs(b)<0.02) %找曲线交点

  • zeros( ) %生成一个全为0的矩阵

    p=zeros(1,101);P=zeros(1,101);
    for z=0:12
    p(z+1)=-1isqrt(2pi/(k1r))A1^2sin(kz1z0)sin(kz1z)-1isqrt(2pi/(k2r))A1^2sin(kz2z0)sin(kz2z);
    P(z+1)=abs(p(z+1));
    end

要注意的地方：

  • 三角函数里的值只能用弧度制表示才能带入计算，如sin(pi/2),cos(pi/6)；若画图中坐标要用角度制表示则应该乘以180/pi做换算
